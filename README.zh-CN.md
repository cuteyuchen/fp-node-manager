# 项目管理器

一款现代化跨平台桌面应用，将前端项目管理、Node.js 版本管理与 Git 仓库管理集于一体。基于 Tauri v2 + Vue 3 + TypeScript 构建。

同时提供 **uTools 插件** 版本。

---

## 功能特性

### 项目管理

通过侧边栏统一管理所有前端项目，可直接运行 `npm`/`yarn`/`pnpm` 脚本（如 `dev`、`build`），实时查看每个脚本的控制台输出。

> 截图占位

- 选择文件夹添加项目，自动识别 `package.json` 中的脚本命令
- 支持 **Node.js 项目**（npm/yarn/pnpm）和**通用项目**（自定义命令）
- 支持**置顶**常用项目
- 一键在编辑器、终端或文件管理器中打开项目
- 支持拼音搜索项目名

---

### Git 管理

内嵌完整 Git 图形界面，采用 SourceTree 风格布局，直接在项目工作区使用。

> 截图占位 — 文件状态视图

**文件状态（变更选项卡）**

- 已暂存 / 未暂存 / 未跟踪文件列表，带状态颜色指示
- 支持暂存、取消暂存、丢弃单个文件或所有文件
- 可调整大小的分割面板（文件列表 | 差异查看器 | 提交区域）
- 基于 diff2html 的语法高亮差异视图

> 截图占位 — 差异查看器

**提交与推送**

- 内置提交信息输入框（切换选项卡不会丢失内容）
- 一键**提交**或**提交并推送**
- 支持 AI 自动生成提交信息（见下方）

> 截图占位 — 提交区域

**提交历史选项卡**

- 完整提交列表，含作者、日期、短哈希、分支/标签引用
- 点击提交可查看变更文件和差异
- 支持加载更多历史记录

> 截图占位 — 提交历史

**分支管理**

- 查看本地和远程分支
- 切换、创建、重命名、删除分支
- 显示与远程分支的领先/落后信息

> 截图占位 — 分支对话框

**远程操作**

- 工具栏一键拉取、推送、获取
- 管理远程仓库地址

---

### AI 提交信息生成

基于暂存区差异，使用任意 OpenAI 兼容接口自动生成提交信息。

> 截图占位

- 可配置 Base URL、API Key 和模型（支持 OpenAI、DeepSeek、通义千问等）
- 默认提示词遵循 **Conventional Commits** 规范，正文使用中文
- 支持完全自定义提示词模板
- 内置接口连通性测试

---

### Node 版本管理

> 截图占位

可视化管理 NVM 安装的 Node.js 版本：

- 安装、卸载、切换 Node.js 版本
- 一键设置**系统默认** Node.js 版本（自动处理环境变量）
- 检测并使用系统已安装的 Node.js（NVM 之外）
- 添加自定义 Node.js 可执行文件

---

### 终端与编辑器集成

- 自动检测系统可用终端（CMD、PowerShell、Git Bash、Windows Terminal、iTerm2 等）
- 支持添加**自定义终端可执行文件**
- 配置首选代码编辑器路径（如 VS Code、Cursor）
- 在项目卡片中一键用编辑器、终端或文件管理器打开项目

---

### 设置

> 截图占位

| 选项 | 可选值 |
|---|---|
| 主题 | 深色 / 浅色 / 跟随系统 |
| 语言 | 简体中文 / English |
| 默认终端 | 自动检测 + 自定义 |
| 编辑器路径 | 任意可执行文件 |
| 自动更新 | 开启 / 关闭 |
| 开机自启 | 开启 / 关闭 |
| 右键菜单 | 资源管理器集成（仅限 Windows） |

支持数据导出/导入，方便备份和迁移。

---

### uTools 插件

本应用同时打包为 **uTools 插件**（`dist-utools/`），可在 uTools 启动器中使用完整的项目管理和 Git 功能，无需独立窗口。

## 技术栈

| 层级 | 技术 |
|---|---|
| 核心 | Tauri v2, Rust |
| 前端 | Vue 3, TypeScript, Vite |
| UI | Element Plus, UnoCSS（兼容 Tailwind） |
| 状态管理 | Pinia |
| 国际化 | vue-i18n |
| 搜索 | pinyin-pro |
| 差异展示 | diff2html |

---

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式（Tauri 桌面端）
npm run tauri dev

# 构建桌面应用
npm run tauri build

# 构建 uTools 插件
npm run build:utools
```
